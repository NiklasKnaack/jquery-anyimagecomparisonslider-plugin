Welcome

Hi, and welcome. I'm glad you're here.
You're probably here because you're looking for a slider to compare images. Very good. Because you've come to the right place. If you do not want to read this welcome text bla bla, you can jump directly to the Quickstart or to the examples.
The any Image Comparison Slider (I know, it's a bulky name) is on the one hand super slim and does not need any additional libraries. On the other hand it is super flexible, versatile and highly customizable. With just a few simple steps, the Slider is ready to use very quickly and runs on almost any device.
In short, it is the last slider you will ever need. And now have fun.

Quickstart

Let's go

If you are in a hurry, this is the perfect place to start. Below you will find an example of the default slider and everything you need to get this thing running.
But if you want to customize your slider and learn more about the possibilities and areas of application, please have a look at the detailed introduction and the examples.

Images: Planet Labs Inc

This is the default slider. If you want to use it directly as shown, just follow the steps below and you will be done in no time.

First, get the jquery.anyimagecomparisonslider.min.js file you need directly from this GitHub Repo page and implement it into the page like your other scripts.

Please note, if you want to use the plugin together with jQuery, then implement the slider script behind jQuery.

HTML 
                            
<script src='./path/jquery.anyimagecomparisonslider.min.js'></script>

Then insert the following code at the position of your page where you want to have the slider. Of course you have to adjust the image paths accordingly. This is done with the data-src attribute.

HTML 
                            
<div class="row">
    <div id="aics">
        <div class="images">
            <div class="image-rgt" data-src="./path/image-rgt.jpg"></div>
            <div class="image-lft" data-src="./path/image-lft.jpg"></div>
        </div>
    </div>
</div>

Almost there. Now all you have to do is initialize the slider. Depending on whether you want to work with jQuery or pure JavaScript, you have the following options.

JavaScript 
                            
new AnyImageComparisonSlider( document.getElementById( 'aics' ) );
jQuery 
                            
$( '#aics' ).anyImageComparisonSlider();

Congrats, you have successfully added your first any Image Comparison slider to your site.
